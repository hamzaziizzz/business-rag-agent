# API Documentation (Premium)

This service exposes an OpenAPI schema generated by FastAPI.

## Interactive Docs

* Swagger UI: `/docs`
* OpenAPI JSON: `/openapi.json`

## Core Endpoints

* `POST /ingest` – ingest raw text documents
* `POST /ingest/files` – upload and ingest files
* `POST /query` – grounded Q&A
* `POST /chat` – multi-turn grounded chat

## Auth

Use `X-API-Key` header when `RAG_API_KEY_MAP` is configured.
